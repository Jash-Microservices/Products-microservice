<h1 align="center">Products Microservice</h1>

## Installation

```bash
$ yarn install
```

## Running the app

1. Set up .env file
2. Execute prisma migration
```bash
$ yarn migrate dev
```
And then run the app:

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
